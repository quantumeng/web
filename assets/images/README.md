# Imágenes Requeridas para Gosth Landing Page

Esta carpeta debe contener las siguientes imágenes para que la landing page funcione correctamente:

## Logos y Branding
- **logo.png** - Logo de la marca (recomendado: 200x60px, formato PNG con transparencia)

## Imágenes Principales
- **hero.png** - Imagen hero de la sección principal (recomendado: 1200x800px o similar)
- **about.jpg** - Imagen de la sección Acerca de (recomendado: 800x600px o similar)

## Iconos de Características
Todos los iconos deben ser de 80x80px (o más grandes, se escalarán automáticamente):
- **icon-tournament.png** - Icono de torneos
- **icon-ranking.png** - Icono de ranking/clasificación
- **icon-stream.png** - Icono de streaming
- **icon-team.png** - Icono de equipos
- **icon-coach.png** - Icono de coaching/entrenamiento
- **icon-analytics.png** - Icono de analíticas/estadísticas

## Imágenes de Torneos
Todas las imágenes de torneos deben ser de aproximadamente 600x400px:
- **tournament-1.jpg** - Imagen del torneo principal (Mega Championship)
- **tournament-2.jpg** - Imagen del torneo semanal
- **tournament-3.jpg** - Imagen del torneo diario

## Notas Importantes
- Todas las imágenes deben estar optimizadas para web
- Formatos recomendados: PNG para logos/iconos (con transparencia si es necesario), JPG para fotografías
- Puedes usar imágenes placeholder mientras consigues las imágenes finales
- Las imágenes se ajustarán automáticamente con CSS, pero es recomendable usar los tamaños sugeridos

## Alternativas Rápidas
Si necesitas imágenes placeholder rápidas, puedes usar:
- [Unsplash](https://unsplash.com/) - Para fotografías de alta calidad
- [Pexels](https://www.pexels.com/) - Para imágenes gratuitas
- [Flaticon](https://www.flaticon.com/) - Para iconos
- [Canva](https://www.canva.com/) - Para crear logos simples
